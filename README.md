# SystemDesign (LLD in Java)

This repository contains **Low Level Design (LLD)** implementations in **Java**, focusing on **Object-Oriented Programming (OOP)** concepts, **SOLID principles**, and commonly used **Design Patterns**.

It is intended as a **learning + interview-prep repository** for backend / SDE roles.
